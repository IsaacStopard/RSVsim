# Summary function to calculate the age-specific total incidence over the simulation.

Function calculates the total incidence stratified by age over the whole
simulation (excluding warm up).

## Usage

``` r
RSVsim_total_incidence(out)
```

## Arguments

- out:

  `RSVsim_run_model` function output.

## Value

Summary statistics (total incidence ordered by age-group) from the
simulation output (dataframe).
