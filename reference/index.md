# Package index

## All functions

- [`RSVsim`](https://isaacstopard.github.io/RSVsim/reference/RSVsim.md)
  : RSVsim: A package for simulating Respiratory Synctial Virus (RSV)
  transmission.
- [`RSVsim_ABC_SMC()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_ABC_SMC.md)
  : Function to run an Approximate Bayesian Computation Sequential Monte
  Carlo (ABC-SMC) algorithm
- [`RSVsim_ABC_rejection()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_ABC_rejection.md)
  : Function to run an Approximate Bayesian Computation (ABC) rejection
  algorithm
- [`RSVsim_abs_dist_fun()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_abs_dist_fun.md)
  : Distance function to calculate the absolute difference between two
  vectors.
- [`RSVsim_amplitude()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_amplitude.md)
  : Summary function to calculate the age-specific amplitude of changes
  in incidence over the simulation.
- [`RSVsim_contact_matrix()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_contact_matrix.md)
  : Create contact matrix for RSVsim
- [`RSVsim_parameters()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_parameters.md)
  : Get parameters for RSVsim
- [`RSVsim_peak()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_peak.md)
  : Summary function to calculate the age-specific timing of peak
  incidence.
- [`RSVsim_run_model()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_run_model.md)
  : Runs the RSV model using odin and dust
- [`RSVsim_shortest_periodic_dist_fun()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_shortest_periodic_dist_fun.md)
  : Distance function to calculate the shortest periodic difference
  between two vectors.
- [`RSVsim_total_incidence()`](https://isaacstopard.github.io/RSVsim/reference/RSVsim_total_incidence.md)
  : Summary function to calculate the age-specific total incidence over
  the simulation.
